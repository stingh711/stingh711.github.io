To make file field's URL right in Django rest framework

11 Jul 2018

I have a django model which has a fiel field and I try to use a ModelSerializer to serialize the model to return the file’s access URL. However, the default behavior seems wrong, it will return an absolute URL base on the URL of current request. After some google work, I find the answer on stackoverflow. What I need is SerializerMethodField. It will generate a field’s content with a method.

class FileSerializer(serializers.ModelSerializer):
    file_url = serializers.SerializerMethodField('get_file_absolute_url')

    def get_file_absolute_url(self, obj):
    return obj.file.url

    class Meta:
    model = File
    fields = ('file', 'id', 'name', 'file_url')
    read_only_fields = ('id', 'name', 'file_url')

Now the file_url will return the right URL.
