How to remove vscode's decorator warning?

Posted on Jun 19, 2018
  • Create a tsconfig.json under root directory.
  • Add following code to tsconfig.json
1
2
3
4
5
6

   {
       "compilerOptions": {
           "experimentalDecorators": true,
           "allowJs": true
       }
   }
  • Restart vscode
