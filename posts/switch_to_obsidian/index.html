Switch to obsidian

Posted on Oct 27, 2020

I used to use bear as my note taking app. However, as the roam reseach began to get popular, I switched to org-roam, a great emacs extension. It is base on org mode, so the edit feature is really powerful.

However, it is still an editor, not an delicated note app. Some features are not easy to use. For example, seaching or just skiming some notes. The graphing feature of roam research is also needing improving.

Then I meet obsidian. It is just like an enhanced Bear with roam features. It has all the features I want. Linking, back linking, powerful searching and graphing. All files are saved in plain markdown files (I write this post in obsidian and write a simple script to convert to hugo’s format. It even has a vim plugin.

However, having a good app is one thing, keeping good note is another thing. Zettelkasten and roam research are good methods to keep notes, but the content is still the king.
