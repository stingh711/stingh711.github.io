How to remove vscode's decorator warning?

Posted on Tue 19 June 2018 in editor • Tagged with vscode, editor, javascript

  1. Create a tsconfig.json under root directory.
  2. Add following code to tsconfig.json
   {
       "compilerOptions": {
           "experimentalDecorators": true,
           "allowJs": true
       }
   }
  1. Restart vscode
